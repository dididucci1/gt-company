# 🚀 Guia Rápido - Novas Funcionalidades

## 🎨 Mudança de Tema (Light/Dark)

### Como ativar o Dark Mode:
1. **Localize o ícone** no header (canto superior direito)
   - 🌙 Lua = ativar dark mode
   - ☀️ Sol = ativar light mode

2. **Clique no ícone** para alternar

3. **Sua preferência é salva automaticamente!**

## 📊 Contas a Pagar - Visualização por Fornecedor

### Visão Geral
Agora você vê as contas **agrupadas por fornecedor** com estatísticas!

### Como usar:

#### 1️⃣ Na lista de fornecedores você vê:
```
┌────────────────────────────────────────┐
│ > Construtora Silva & Filhos           │
│   12.345.678/0001-90 • 5 conta(s)     │
│                                         │
│   ⏰ 3 pendentes     ✅ 2 pagas        │
│   R$ 85.000,00      R$ 50.000,00       │
└────────────────────────────────────────┘
```

#### 2️⃣ Clique no fornecedor para expandir
- **Seta vira para baixo** 🔽
- **Mostra todas as contas** do fornecedor

#### 3️⃣ Veja os detalhes de cada conta:
- ✏️ Descrição
- 🏷️ Categoria
- 💰 Valor
- 📅 Data de vencimento
- 🟢 Status (Pago/Pendente/Atrasado)
- ⚠️ Alertas (se tiver atraso)

#### 4️⃣ Para recolher:
- Clique novamente no fornecedor

### Dica de uso:
📌 Use o **filtro** no topo para buscar um fornecedor específico!

## 🆘 Resolução de Problemas

### Tema não muda?
1. Limpe o cache do navegador
2. Recarregue a página (F5)

### Fornecedor não expande?
1. Certifique-se que clicou no card inteiro
2. Recarregue se necessário

### Imagens não aparecem no login?
1. Verifique se as imagens estão em `/public`
2. As imagens devem se chamar `logo.png` e `image.png`
