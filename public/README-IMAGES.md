# Imagens para o Login

Coloque as seguintes imagens nesta pasta:

## Arquivos necessários:

1. **logo.png** - Logotipo da GT Company (recomendado: fundo transparente, 400x200px)
2. **image.png** - Imagem de fundo do painel lateral (recomendado: 1920x1080px, imagem de obra/construção)

## Formatos aceitos:
- PNG (recomendado para logo)
- JPG/JPEG (recomendado para imagem de fundo)
- WebP (otimização opcional)

## Dicas:
- Use imagens de boa qualidade
- Logo com fundo transparente funciona melhor
- Imagem de fundo pode ser de uma obra da empresa
- Dimensões sugeridas mantêm boa resolução em telas HD

## Organização:
```
/public
  /logo.png         # Logo da empresa
  /image.png        # Imagem de fundo
```

Após adicionar as imagens, elas serão automaticamente carregadas na tela de login.
